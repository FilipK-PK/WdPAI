# Aplikacja do sprzedawania i kupowania lokalnych rękodzieł.
* Działa na zasadzie lokalizacji.
* Użytkownik może wyszukiwać pracę w określonej odległości od siebie.
* Może też wyszukiwać konkretnych prac np.: obrazy, rzeźby.
* Istnieje możliwość kontaktu z artystą, przed kupnem lub zwykłym zapytaniem się.
##### Posiadam pełne prawa autorskie do przedstawionych prac i zdjęć prac, ponieważ jestem ich autorem.
